Hello World!

Ini adalah tulisan pertama saya di blog ini. Blog ini dibuat dengan Hugo dan di deploy ke Github Pages. Kenapa menggunakan Hugo? Karena saya terinspirasi dari blog Petanikode.com yang kebetulan saat itu sedang migrasi ke Hugo, lalu saya coba coba untuk membuat projek dari Hugo.. eh ternyata gampang juga ya. Lalu setelah mengumpulkan niat saya langsung coba implementasikan untuk membuat blog ini dengan tutorial dari dokumentasi resmi Hugo dan dari blog Petanikode.com.

Blog ini nantinya akan membahas seputar dunia programming khususnya yaitu web programming. Mohon maaf kalau untuk tata bahasanya masih belum sebagus blog blog yang lain karena ini memang baru pengalaman pertama saya membuat dan menulis di blog.

^_^
